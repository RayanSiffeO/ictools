

# ictools 🚀

R tools for computing confidence intervals from data frames and matrices, including classical and bootstrap methods
[![R version](https://img.shields.io/badge/R-4.3-blue.svg)](https://www.r-project.org/) 
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)
[![Build Status](https://img.shields.io/badge/build-passing-brightgreen.svg)]()

**ictools** es un paquete de **R** diseñado para simplificar el análisis de **series temporales interrumpidas (ITS)** y otros métodos estadísticos avanzados. Este paquete permite realizar análisis reproducibles, visualizaciones claras y pipelines de datos eficientes, todo listo para aplicaciones en ciencia de datos y análisis de impacto.

---

## 🌟 Características principales

- Análisis de **series temporales interrumpidas** (ITS)
- Modelos estadísticos aplicables a datos longitudinales
- Visualización integrada y fácil de interpretar
- Funciones **modulares y reutilizables**
- Ejemplos reproducibles para aprendizaje rápido
